# Homework

Daily assignments created by Hobson and completed by Dan.

## Format

- One file per assignment, named by date: `2026-03-24.md`
- Each file contains the assignment with clear instructions
- Dan writes his answers directly in the file under each prompt
- Hobson reviews and adds corrections in the next session

## Types

- **Grammar drills:** Conjugation, sentence transformation, fill-in-the-blank
- **Writing assignments:** Paragraphs or short essays on a given topic, in Spanish
- **Reading comprehension:** A passage in Spanish followed by questions
- **Translation:** English → Spanish or Spanish → English

## Rules

- Do the assignment the day it's assigned
- Commit your completed work to the repo
- **Grammar drills:** Open book. Check Anki cards or reference material as needed. The point is producing correct forms, not testing memory.
- **Writing assignments:** Closed book. No translators, no references. Use what's in your head.
- **Exams:** Always closed book unless stated otherwise.
- If you're stuck, attempt it anyway — wrong answers are more useful than blank ones
