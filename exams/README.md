# Exams

Periodic assessments created by Hobson to measure progress and calibrate instruction.

## Schedule

- **Baseline diagnostic:** First exam. Establishes actual current level across all skills.
- **Progress exams:** Every 2-4 weeks, covering material from recent homework and Anki batches.
- **Phase exit exams:** Before advancing to the next phase.

## Format

- One file per exam: `baseline.md`, `week04.md`, etc.
- Sections: grammar, vocabulary, reading comprehension, writing, translation
- Dan completes the exam in the file and commits it
- Hobson grades it and adds scores + corrections
- Results are recorded in the progress log in `spanish-framework.md`

## Rules

- No references, no translators, no dictionaries unless specified
- Complete in one sitting
- Time yourself and note the duration
